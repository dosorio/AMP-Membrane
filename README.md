cAMP-Membrana
=============

**Interacciones de cAMP's con Membranas**
Código de Simulaciones entre Péptidos Catiónicos Antimicrobianos y Bicapas lípidicas puras de POPE, POPG y POPC para identificar secuencias biológicamente activas
